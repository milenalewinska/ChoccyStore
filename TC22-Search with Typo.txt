TC-22: Search with Typo

1. Objective:
This test case verifies that User enters a slightly misspelled chocolate name.

2. Environment : ChoccyStore application.

3. Version: Demo 1.1.1001

4. Preconditions:
a. The application is up and running.
b. The system supports fuzzy search or spelling suggestions.

5. Test Steps:
a. Open a web browser and navigate to the online chocolate shop's login page.
b. Navigate to the search bar.
c. Enter "Dak Chocolate" (misspelling).
d. Press the "Search" button or hit "Enter."


6. Expected Result: 
a. The system suggests "Did you mean 'Dark Chocolate'?" or returns results for "Dark Chocolate."
