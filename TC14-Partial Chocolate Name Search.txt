TC-14: Partial Chocolate Name Search

1. Objective:
This test case verifies that User searches using a partial chocolate name.

2. Environment : ChoccyStore application.

3. Version: Demo 1.1.1001

4. Preconditions:
a. The application is up and running.
b. Partial names (e.g., "Milk") match multiple chocolate names.

5. Test Steps:
a. Open a web browser and navigate to the online chocolate shop's login page.
b. Navigate to the search bar.
c. Enter the partial chocolate name, "Milk."
d. Press the "Search" button or hit "Enter."


6. Expected Result: 
a. All chocolates with names containing "Milk" are displayed (e.g., "Milk Chocolate", "Milk and Nuts Chocolate").
