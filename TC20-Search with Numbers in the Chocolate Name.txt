TC-20: Search with Numbers in the Chocolate Name

1. Objective:
This test case verifies that User searches for chocolates whose names contain numbers.

2. Environment : ChoccyStore application.

3. Version: Demo 1.1.1001

4. Preconditions:
a. The application is up and running.
b. Some chocolates in the database have names with numbers (e.g., "70% Dark Chocolate").

5. Test Steps:
a. Open a web browser and navigate to the online chocolate shop's login page.
b. Navigate to the search bar.
c. Enter "70%" in the search bar.
d. Press the "Search" button or hit "Enter."


6. Expected Result: 
a. Chocolates with "70%" in their names are displayed (e.g., "70% Dark Chocolate").
