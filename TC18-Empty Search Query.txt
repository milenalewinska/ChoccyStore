TC-18: Empty Search Query

1. Objective:
This test case verifies that User clicks the search button without entering any search term.

2. Environment : ChoccyStore application.

3. Version: Demo 1.1.1001

4. Preconditions:
a. The application is up and running.
b. Search bar is empty.

5. Test Steps:
a. Open a web browser and navigate to the online chocolate shop's login page.
b. Navigate to the search bar.
c. Leave the search bar empty.
d. Press the "Search" button or hit "Enter."


6. Expected Result: 
a. A message or a default list of all chocolates should be displayed (or a message like "Please enter a search term" if applicable).
