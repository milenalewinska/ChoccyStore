TC-14: Uppercase Search

1. Objective:
This test case verifies that User searches with different cases upper case for the chocolate name.

2. Environment : ChoccyStore application.

3. Version: Demo 1.1.1001

4. Preconditions:
a. The application is up and running.
b. Navigate to the search bar.
c. Enter "DARK CHOCOLATE" .
d. Press the "Search" button or hit "Enter.

5. Test Steps:
a. Open a web browser and navigate to the online chocolate shop's login page.
b. Navigate to the search bar.
c. Enter the partial chocolate name, "Milk."
d. Press the "Search" button or hit "Enter."


6. Expected Result: 
a. Results are the same regardless of letter case. The same chocolates should be displayed.
